## 实现方案

### 设计原则
1. **最小侵入**：只新增一个独立的运行脚本，不修改现有代码
2. **复用现有组件**：充分利用现有的审计器、解析器等核心组件
3. **保持一致性**：与现有模板保持相同的使用方式和输出格式
4. **高性能**：默认使用高性能审计器（FastAuditor）

### 核心功能

#### 1. CSV文件读取
```python
def read_violation_csv(file_path: str) -> pd.DataFrame:
    # 跳过第二行（中文列名）
    df = pd.read_csv(file_path, encoding='utf-8-sig', skiprows=[1])
    return df
```

#### 2. 字段映射
| 违规使用模板 | 审计器需要 | 说明 |
|------------|----------|------|
| `sub_acct_name` | account | 从账号名称 |
| `client_ip` | source_ip | 本端IP（源IP） |
| `server_ip` | target_ip | 对端IP（目标IP） |
| `op_fort_content` | operation_content | 操作内容 |

#### 3. SSH命令构造
```python
# 如果操作内容不是SSH命令格式，自动构造
if not operation_content.strip().startswith('ssh'):
    constructed_cmd = f"ssh {account}@{server_ip}"
```

#### 4. 时间验证
- 自动从文件名提取日期（如：20260203）
- 使用文件日期进行时间范围验证

## 三种模板对比

| 特性 | 普通模板 | 4A模板 | 违规使用模板 |
|------|---------|--------|------------|
| **运行脚本** | `run_audit.py` | `run_audit_4a.py` | `run_audit_violation.py` |
| **列名语言** | 中文 | 中文 | 英文 |
| **第二行** | 无 | 无 | 中文说明（跳过） |
| **源IP字段** | 资源IP | 源地址 | client_ip |
| **目标IP字段** | 从SSH提取 | 目的地址/SSH | server_ip |
| **账号字段** | 从账号/主账号 | 从账号/主账号 | sub_acct_name |
| **命令构造** | 不需要 | 不需要 | 可能需要 |
| **特殊处理** | IP提取 | 堡垒机跳板 | 命令构造 |
