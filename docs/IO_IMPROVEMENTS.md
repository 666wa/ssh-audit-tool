# 输入输出优化总结

## 优化内容

### 1. 输出文件命名优化
**新增功能：**
- ✅ 支持自定义输出路径（`--output`参数）
- ✅ 支持时间戳选项（`--no-timestamp`参数）
- ✅ 自动生成规范的输出文件名

**命名规则：**
```
默认：结果-<原文件名>.xlsx
时间戳：结果-<原文件名>_20251225_191241.xlsx
自定义：用户指定的任意路径和文件名
```

**示例：**
```bash
# 默认命名
python run_audit.py -o 操作表.xlsx -r 报备表.xlsx
# 输出：结果-操作表.xlsx

# 自定义输出
python run_audit.py -o 操作表.xlsx -r 报备表.xlsx --output 我的结果.xlsx
# 输出：我的结果.xlsx

# 带时间戳（需要不使用--no-timestamp）
python run_audit.py -o 操作表.xlsx -r 报备表.xlsx
# 输出：结果-操作表.xlsx（默认不带时间戳）
```

## 使用示例

### 基本用法

## 技术实现

### 1. 进度显示算法
```python
# 智能计算进度间隔
progress_interval = max(1, total // 20)  # 显示20次进度

# 自适应显示
if (idx + 1) % progress_interval == 0 or (idx + 1) == total:
    percentage = (idx + 1) / total * 100
    logger.info(f"进度: {idx + 1}/{total} ({percentage:.1f}%)")
```

### 2. 文件大小格式化
```python
def format_file_size(size_bytes: int) -> str:
    for unit in ['B', 'KB', 'MB', 'GB']:
        if size_bytes < 1024.0:
            return f"{size_bytes:.2f} {unit}"
        size_bytes /= 1024.0
    return f"{size_bytes:.2f} TB"
```

### 3. 时间戳生成

