# 数据格式参考

## 报备表（所有模板共用）

必需列：
- `访问账号` — 可包含多个账号，以逗号/顿号分隔
- `本端主机IP` — 支持精确IP、IP段、CIDR、多IP混合（顿号分隔）
- `对端主机IP` — 同上

可选时间列：
- `生效时间` — 报备生效日期
- `失效时间` — 报备失效日期

注意：报备表CSV第一行可能是说明行（包含"允许录入多个"等文字），需跳过。

## 标准SSH操作命令表

必需列：`操作时间`、`资源IP`、`操作内容`、`主账号名称`
可选列：`从账号名称`（优先使用）、`资源名称`（备用IP来源）

## 4A操作命令表

必需列（英文）：`op_fort_content`、`res_ip`、`main_acct_name`
可选列：`sub_acct_name`、`res_name`

## 账号违规使用表

必需列：`sub_acct_name`、`client_ip`、`server_ip`、`op_fort_content`、`op_time`
实际匹配使用：`server_ip`（本端）、`dst_ip`（对端）
特殊：第二行是中文列名，读取时需 skiprows=[1]

## IP格式示例

| 格式 | 示例 |
|------|------|
| 精确IP | `10.246.85.123` |
| IP段 | `10.230.72.1-10.230.72.254` |
| 多段连续 | `10.230.72.1-10.230.72.254-10.230.73.1-10.230.73.254` |
| CIDR | `192.168.1.0/24` |
| 多IP混合 | `10.1.1.1、10.2.2.0/24、10.3.3.1-10.3.3.100` |

分隔符为中文顿号 `、`。
